# Produto

## Tree Marketplace B2C

Este é um projeto educacional focado no aprendizado de Vue.js e implementação prática de algoritmos matemáticos famosos. O projeto combina desenvolvimento frontend moderno com conceitos matemáticos avançados para criar uma experiência interativa de geração procedural de árvores.

### Funcionalidades Principais

- **Geração Procedural de Árvores**: Utiliza algoritmos matemáticos como Perlin Noise, geradores de números aleatórios com seed e geometria 3D
- **Visualização 3D**: Renderização em tempo real usando Three.js com controles de câmera interativos
- **Sistema de Seeds**: Permite reproduzir árvores idênticas usando códigos alfanuméricos
- **Marketplace Virtual**: Interface para visualizar e "comprar" diferentes tipos de árvores geradas
- **Temas Dinâmicos**: Suporte a modo claro e escuro com persistência de preferências

### Objetivos Educacionais

- Aprender Vue.js 3 com Composition API
- Implementar algoritmos matemáticos famosos (Perlin Noise, L-Systems, geometria fractal)
- Trabalhar com renderização 3D no navegador
- Praticar arquitetura de componentes Vue
- Desenvolver interfaces responsivas e acessíveis
