# Stack Tecnológica

## Framework e Build System

- **Vue.js 3.5.26**: Framework principal usando Composition API
- **Vite 7.3.0**: Build tool e dev server
- **Vue Router 4.6.4**: Roteamento SPA
- **Node.js**: Versões suportadas ^20.19.0 || >=22.12.0

## Bibliotecas Principais

- **Three.js 0.182.0**: Renderização 3D e geometria
- **Lucide Vue Next 0.562.0**: Ícones SVG

## Ferramentas de Desenvolvimento

- **@vitejs/plugin-vue**: Plugin oficial Vue para Vite
- **vite-plugin-vue-devtools**: DevTools integradas

## Algoritmos Matemáticos Implementados

- **Perlin Noise**: Geração de ruído procedural para variações naturais
- **Seeded Random**: Gerador de números pseudo-aleatórios determinístico
- **Geometria Fractal**: Estruturas de ramificação recursiva
- **Transformações 3D**: Rotações, escalas e posicionamento espacial

## Comandos Comuns

```bash
# Desenvolvimento
npm run dev

# Build para produção
npm run build

# Preview da build
npm run preview
```

## Estrutura de Renderização 3D

- **WebGL**: Renderização acelerada por hardware
- **Shadow Mapping**: Sombras dinâmicas
- **Material System**: Materiais Phong para iluminação realística
- **Geometry Instancing**: Otimização para múltiplos objetos similares

## Padrões de Performance

- **Async Tree Building**: Geração não-bloqueante com feedback visual
- **Memory Management**: Dispose adequado de geometrias e materiais
- **LOD System**: Diferentes níveis de detalhe baseados em complexidade
